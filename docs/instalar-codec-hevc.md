# Instalar Codec HEVC

:::info Objetivo
Permite usar o codec nativo do Windows sem necessitar usar um player externo de vídeo.
:::

Fontes: [GitHub](https://gist.github.com/rajibdpi/218be47545f312cc47e577b36e56f5c7)

## Guia

1. Acesse o [site](https://store.rg-adguard.net/)

2. Selecione `ProductID` e cole `9n4wgh0z6vhq` e pressione ✔ para procurar

3. Selecione a opção desejada e clique com o botão direito em `Salvar link como...`

4. Execute o arquivo .appx para instalar
